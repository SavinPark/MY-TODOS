# MY-TODOS
HTML, CSS 그리고 Javascript로 만든 투두리스트입니다.

## 기능
- 날짜 및 시간 표현
- 등록
- 완료/미완료 상태 표시
- 수정
- 삭제

## 구조
### Header
- year, month, date
- day, time(hour:minute)
### Main -- todos
- 할 일
- 완료/미완료 아이콘
- 수정 아이콘
- 삭제 아이콘
### Main -- form
- 제목
- 내용
### Footer -- todos
- 추가 버튼 ( + )
### Footer -- form
- 저장 버튼 ( save )
- 취소 버튼 ( x )
